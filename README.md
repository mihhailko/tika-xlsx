# tika-xlsx

Compile Apache Tika json files to desirable xlsx format

## Installation

    lein uberjar

## Usage

    $ java -jar tika-xlsx-0.1.0-standalone.jar [filename]

## Options

[filename] - desired .xlsx output file path
